# 🎵 音程検出器 - Pitch Detector (Next.js)

マイクから入力された音声をリアルタイムで解析し、音程（ピッチ）を検出・表示するNext.jsアプリケーションです。

## 機能

- **リアルタイム音程検出** - 自己相関法（Autocorrelation）アルゴリズムで高精度に音程を検出
- **ドレミ表記** - 音名を日本語のドレミで表示
- **音階グラフ** - 検出した音程の推移をリアルタイムでグラフ表示
- **チューナーメーター** - 正確な音程からのズレをセント単位で視覚化
- **音量レベル表示** - 入力音量をリアルタイムでバー表示
- **ピアノキーボード** - 現在検出中の音をキーボード上でハイライト

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで http://localhost:3000 を開いてください。

## 使い方

1. 「開始」ボタンをクリック
2. マイクへのアクセスを許可
3. 楽器を鳴らすか、声を出す
4. リアルタイムで音程が表示されます

## 技術スタック

- Next.js 15 (App Router)
- React 19
- Web Audio API
- 自己相関法によるピッチ検出

## プロダクションビルド

```bash
npm run build
npm start
```
